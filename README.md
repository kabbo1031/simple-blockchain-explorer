# Simple Blockchain Explorer

This is a simple blockchain project for beginners.

This project helps you:
- Connect to a blockchain network
- View latest block number
- View block transaction count
- Learn how block data works

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- Internet connection
- Basic JavaScript knowledge

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Click "Load Blockchain Data"

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Network settings  
- `explorer.js` = Block reader  
- `ui.js` = Screen updates  
- `provider.js` = Blockchain provider  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs  
- `package.json` = Project info
